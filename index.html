Retro Readers

A fun and interactive experience for children with dyslexia to learn how to read and spell.

Read more

Retro Readers is a brain training game

Using OpenDyslexic, an open-source font designed to specifically lessen reading errors made by readers diagnosed with dyslexia.

Responsive

Retro Readers utilizes the random-word python module which incorporates Wordnik API for its reading gamemode. This results in a complex and efficient library of words of various difficulty for the reader to learn.

Inspiration

Retro Readers was inspired by a social media post pertaining to another text font that was proven to help mitigate reading errors made by dyslexic readers.

Fulfilling

Gameplay is setup in a way that players/readers can benefit using the in game currency to purchase customizable equipment and visual modifications.
